🚀 Getting Started

1) Quick Setup (Recommended)

Use convenience scripts for automated setup.
# Windows
npm run setup:win
npm run ml:setup:win

# Unix/Linux/macOS
npm run setup:unix
npm run ml:setup:unix

2) Manual Setup

Step-by-step manual configuration.
npm install
cp .env.simplified.example .env
# Edit .env with your API keys

3) Initialize Database

Setup Supabase with knowledge substrate and ML schemas.
# Automated (Recommended)
npm run init-db:win  # or init-db:unix
npm run ml:init-db:win  # or ml:init-db:unix

# Manual
# Run init-knowledge-substrate.sql in Supabase
# Run db/supabase/ml_schema.sql in Supabase

4) Start Services

Launch with Docker or local development.
# Docker (Recommended)
npm run docker:simplified

# Local Development
npm run dev
npm run ml:service  # Start ML service separately

5) Test ML Service

Validate ML service functionality and endpoints.
# Test ML Service
npm run ml:test:win  # or ml:test:unix

# Access ML API Documentation
# http://localhost:8001/docs
